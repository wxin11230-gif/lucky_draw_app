# 🎡 Lucky Draw App

An interactive, web-based random selection tool featuring smooth animations and sound effects.

## 🚀 Live Demo
You can try the application here: [lucky-draw-app-gules.vercel.app](https://lucky-draw-app-gules.vercel.app/)

## ✨ Features
* **Random Selection Logic:** Fair and unbiased picking powered by JavaScript.
* **Audio Experience:** Includes custom sound effects (`shake.mp3` and `success.mp3`) for excitement.
* **Clean UI:** Styled with modern CSS for a sleek look.
* **Instant Deployment:** Hosted on Vercel for high availability.

## 🛠️ Built With
* **JavaScript** - The engine behind the lucky draw.
* **CSS3** - For the layout and animations.
* **HTML5** - The structural foundation.
